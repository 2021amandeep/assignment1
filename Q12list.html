A Nested List

Lists can be nested (list inside list):

  1. Fruits
  2. Vegetable
    1. Potato
    2. Tomato
      1. Potato
      2. Tomato
  3. Shoes
  1. li element
    1. sub li element
    2. sub li element
    3. sub li element
  2. li element
  3. li element
    1. sub li element
    2. sub li element
    3. sub li element
